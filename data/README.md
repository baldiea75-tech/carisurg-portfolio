## Dataset

## Week 0 Dataset
The Week 0 dataset used in this project is:

**File:** EmergencyTriageDataset_Reduced_Dirty.csv  
**Rows:** 2,205 patients  
**Columns:** 11  
**Source:** Provided by the CariSurg MedTech Pathways programme for educational use only

The dataset simulates emergency triage records from Mercer General Hospital. It is not included here because it contains simulated patient data. Programme participants can access it via the CariSurg programme portal.

## Week 5 Dataset
The Week 5 dataset used in this project is:

**File:** yaleemmlc_admissionprediction_triage.csv  
**Rows:** 55,121 encounters  
**Columns:** 225  
**Source:** De-identified subset of the Yale EMMLC admission prediction dataset (Hong WS, Haimovich AD, Taylor RA. Predicting hospital admission at emergency department triage using machine learning. PLOS ONE. 2018;13(7):e0201016), publicly available via https://github.com/yaleemmlc/admissionprediction

The dataset contains adult emergency department visits from the Yale New Haven Health System (one academic and two community emergency departments, March 2014 to July 2017), covering triage vital signs, demographics, arrival details, chief complaint flags and the Emergency Severity Index (ESI). It is not included here directly because of its size, but the file is publicly available at the GitHub repository above and citation of the original paper is required for any research use.
