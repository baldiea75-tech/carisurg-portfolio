## Notebooks

### week0_triage_eda.ipynb
Initial exploratory analysis of the triage dataset.

### week0_triage_eda_day2.ipynb
Additional data cleaning and feature exploration.

### week0_triage_eda_day3.ipynb
Final exploratory analysis and visualization work.

### Week_5_Interim_deliverable.ipynb
Exploratory analysis of a new, real emergency medicine dataset (55,121 encounters, 225 columns) to assess its feasibility for baseline model training in Week 6. Defines vital sign, demographic, admin, leakage and chief complaint column families, profiles the ESI target distribution and class imbalance, checks missingness and outliers, examines correlation with ESI, produces four plots and drafts a feasibility memo for the ED Board.
